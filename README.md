# 2048小游戏
  <img src="https://visitor-badge.laobi.icu/badge?page_id=dougunduzii.2048" alt="visitor badge"/>
这是一个使用pygame实现的<strong>丑陋的</strong>2048小游戏GUI版

## 📖 Requirement
pygame

## ❓How to begin?
`git clone https://github.com/dougunduzii/2048`<br>
`pip install pygame`<br>
`python main.py`
使用键盘`↑↓←→`控制，失败请点击`Reset`
